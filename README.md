<h4>Intro</h4>
A Raspberry Pi-based greenhouse project.

<h4>Hardware list:</h4>

- DHT11 - temperature and humidity sensor

- GRL - 12503

- LED diodes powered by 5V

<h4>Setup:</h4>
$ python3 -m pip install adafruit-circuitpython-dht

$ sudo apt-get -y install python3-rpi.gpio
